To run the Web Demo you will need to create a virtual directory
and set permissions to allow your Web user account WRITE ACCESS
to the directory. Usually this is the NETWORK SERVICE account.

The easiest way to configure this is to run the 

_WEBCONFIGURATION.EXE 

file which allows you to create a virtual and set permissions.
